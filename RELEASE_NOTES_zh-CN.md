# Release Notes

## 标题建议

PeepingDHeditor macOS 适配整理版

## 仓库简介建议

适用于 macOS 的 Peeping Dorm Manager 本地存档编辑脚本，包含中文说明、基础隐私清理与 Fork 发布整理。

## Release 文案

本次版本主要完成了针对 GitHub 公开展示与 macOS 使用场景的整理。

更新内容：
- 重写中文 README，补充用途说明、风险提示、Windows 与 macOS 使用方式。
- 突出 macOS 适配定位，支持通过 `PDM_SAVE_PATH` 自定义存档路径。
- 增加 `.gitignore`，避免误提交解密产物、存档文件和常见本地临时文件。
- 增加 `requirements.txt`，简化依赖安装。
- 增加上游来源说明模板，方便在 Fork 后保留原作者信息与修改说明。

注意事项：
- 本项目基于上游仓库整理而来，请保留来源与致谢。
- 上游仓库当前未发现 LICENSE 文件；若需长期公开分发，建议先确认原作者授权状态。
- 本工具仅供学习与交流使用，使用前请自行备份存档。
